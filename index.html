File Size limit is 7MB

Photos taken by smart phone nowadays can easily exceed the limit. Screenshots, on the other hand, could be a good alternative to test out the functionality.

Query

Multiple labels query is available. Each keyword will be automatically separated and placed into the search zone. Keywords in the search zone can be removed by simply clicking on it.

You can either type out the keyword, and then press 'Enter' or spacebar to append to the search zone, or simply click on the default text bubbles.

What exactly is Reko?

Reko is an image analysing tool that is powered by Amazon Rekognition. It can detect labels, faces, emotions, texts, and a lot more. However, the current version only supports label detection.

Label detection enable you to generate metadata for your image libraries for search and filtering.

For future versions, it is planned to implement additional features including:

  • Facial recognition
  • Facial analysis
  • Text in image
